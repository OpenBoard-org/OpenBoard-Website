Web Widgets

In OpenBoard können Sie ganz einfach Web-Widgets erstellen, die auf die Tafel gezogen und abgelegt werden können! Ihre Widgets können sogar mit OpenBoard über eine in OpenBoard integrierte API interagieren! Ein Widget ist einfach eine Website, platziert in einem Ordner mit der Erweiterung .wgt.

Laden Sie die Vorlage herunter, entpacken Sie sie und beginnen Sie, Ihre eigenen Anwendungen zu erstellen: template.wgt.zip (zip, <4KB)

Vergessen Sie nicht, das Symbol (die Datei icon.png) zu ändern, das verwendet wird, um Ihre Anwendung in OpenBoard zu repräsentieren, und füllen Sie die Datei config.xml mit Ihren Informationen und einer eindeutigen Kennung!
