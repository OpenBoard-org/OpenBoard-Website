Widgets Web

Dans OpenBoard, vous pouvez créer des widgets web très facilement, qui pourront ensuite être glissés-déposés sur le tableau ! Vos widgets peuvent même interagir avec OpenBoard, via une API intégrée à OpenBoard ! Un widget est tout simplement un site web, placé dans un dossier avec l'extension .wgt.

Téléchargez le modèle, décompressez-le, et commencez à créer vos propres applications : template.wgt.zip (zip, <4Ko)

N'oubliez pas de modifier l'icone (le fichier icon.png) qui sera utilisé pour représenter votre application dans OpenBoard, et de renseigner le fichier config.xml avec vos informations et un identifiant unique !
