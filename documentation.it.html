Widget Web

In OpenBoard, puoi creare facilmente widget web che possono essere trascinati e rilasciati sulla lavagna! I tuoi widget possono addirittura interagire con OpenBoard attraverso un'API integrata in OpenBoard! Un widget è semplicemente un sito web, posizionato in una cartella con l'estensione .wgt.

Scarica il modello, decomprimilo e inizia a creare le tue applicazioni: template.wgt.zip (zip, <4KB)

Non dimenticare di modificare l'icona (il file icon.png) che verrà utilizzata per rappresentare la tua applicazione in OpenBoard, e di compilare il file config.xml con le tue informazioni e un identificativo unico!
