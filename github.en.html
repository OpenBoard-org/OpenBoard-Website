GitHub

OpenBoard uses GitHub to share its source code. The platform also serves as a place for interaction, where OpenBoard users can participate.

GitHub is an international platform with participants from various nationalities. If you wish to contribute, please communicate in English whenever possible, so that everyone can benefit from your exchanges!

Contribute

You can contribute to the development of OpenBoard by reporting bugs, suggesting improvements, or making changes to the source code yourself. If you are a developer, you can access the source code by following this link.

You will also find wiki pages to help you build your own version of OpenBoard.
