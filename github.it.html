GitHub

OpenBoard utilizza GitHub per condividere il suo codice sorgente. La piattaforma funge anche da luogo di scambio, da cui gli utenti di OpenBoard possono partecipare.

GitHub è una piattaforma internazionale in cui partecipano molte nazionalità. Se desiderate partecipare, comunicate preferibilmente in inglese, in modo che tutti possano beneficiare dei vostri scambi!

Partecipare

Potete contribuire allo sviluppo di OpenBoard segnalando bug, proponendo miglioramenti o apportando modifiche dirette al codice sorgente. Se siete sviluppatori, potete accedere al codice sorgente seguendo questo link.

Troverete anche pagine wiki per aiutarvi a costruire la vostra versione di OpenBoard.
